# Security Playbooks for SMB

Actionable step-by-step guides for common security operations. Designed for small teams without dedicated security staff.

## Available Playbooks

| Playbook | Purpose | When to Run |
|----------|---------|-------------|
| **[Incident Response](incident-response/)** | Handle security incidents | When alerts trigger |
| **[Backup Validation](backup-validation/)** | Verify backup integrity | Monthly |
| **[Patch Management](patch-management/)** | Safe software updates | Bi-weekly |
| **[Weekly Review](weekly-security-review/)** | Routine security checks | Every Friday |

## Usage

1. **Follow checklists sequentially**
2. **Document completion** (owner + date)
3. **Adapt to your environment**
4. **Test first on non-production**

**Philosophy**: Simple → Repeatable → Automated

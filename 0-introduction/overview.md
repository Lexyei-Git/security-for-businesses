# Overview

This guide helps small and medium businesses improve their cyber security posture using clear, practical steps that non-security analysts can follow.

You can navigate this repository as follows:
- Start here to understand the purpose and structure.
- Go to `1-foundations/` to learn basic concepts.
- Go to `2-where-to-start/` to see a 30-day action plan.
- Use `3-practical-controls/` when you want to implement specific measures.

# Nmap Overview

Nmap (Network Mapper) is a free and open source utility for network discovery and security auditing.

It is used to discover hosts and services on a computer network by sending packets and analyzing the responses.

For small and medium businesses, Nmap provides essential network visibility:
- Host discovery across subnets
- Service/version detection
- Operating system fingerprinting
- Vulnerability scanning via NSE scripts
- Network mapping and topology discovery

Nmap runs locally (no server/agent) and supports all major platforms. It's the foundation for network reconnaissance and basic security assessments.
